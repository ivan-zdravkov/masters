Ivan Zdravkov Master's Thesis

Comparing 2D and 3D methods for 3D skeletal model mapping

The master's degree is a result of an 8 month research in the Ishikawa Senoo Laboratory of the Tokyo University in Japan. The full graduation thesis in bulgarian can be downloaded here: Дипломна работа магистър - Иван Здравков.docx

Formulation of the problem

The task is to compare the results of a 2D and 3D methods for 3D skeletal model mapping.

Architecture

Result showcase: An Angular application, querying the zipped images from the server to show the results.
Data gathering: A C# and a MATLAB applications using the Nuitrack SDK and the VNect respectively, owned by the Ishikawa Senoo Laboratory.
